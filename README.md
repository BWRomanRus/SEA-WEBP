# SEA WEBP
Минималистичный модуль чтения статичных и анимированных Webp изображений, без внешних зависимостей

## Назначение
Быстрое чтение .webp всех типов.
Использование в любых проектах Delphi

## Возможности
- Один модуль pas без зависимостей  
- Без библиотек и runtime  

## Недостатки
Бета
